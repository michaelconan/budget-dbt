# Streamlit Dashboard

The project includes a Streamlit application for interactive data exploration and visualization.

## Running the Dashboard

To start the dashboard, run:
```bash
make streamlit-run
```

## Features

- **Spend Overview**: Track total spending over time.
- **Categorical Analysis**: Break down expenses by category (Groceries, Travel, etc.).
- **Vendor Deep-dive**: Identify top vendors and spending patterns.
- **Monthly Comparisons**: Compare spending across different months and years.
